This package provides some tools to work with [Matplotlib](https://matplotlib.org/).

- `freshfig`
- `anchor_axes`, with auxiliaries:
  - `align_ax_with`
  - `set_ax_size`
  - `get_legend_bbox`
- `add_log_toggler`
  - `toggle_scale`

## Installation

#### Normal installation

```sh
pip install mpl-tools
```

#### For development

```sh
git clone <this repo>
make
```
